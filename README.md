#install & service start

1. mongodb 설치
   backend/.env 파일 **_SETTING_** 부분에 환경에 맞게 설정
   MONGO*URI=mongodb://\*\*\_SETTING*\*\*

2. frontend 설치 및 빌드
   cd frontened
   yarn install
   yarn build

3. backend 설치 및 빌드 후 서비스 시작
   cd backend
   yarn install
   yarn start
